Welcome to the zoo database!

This site provides an interface to view, add, edit, and delete items in the database. It is broken up by pages concerning the different tables, which track attributes and relationships. Below you can find a description of, and link to, each section.

Animals

This page allows you to view information about animals in the zoo, as well as add a new animal to the database, delete an animal, and update an animal's pregnancy status.

Animal Foods

This page provides a table of the foods the zoo animals eat, and allows the user to add a new food as well as delete a food from the database.

Employees

This page allows you to view information about the zoo's employees, as well as add a new employee to the database and delete employees.

Zones

This page shows information about the various zones in the zoo, allowing the user to add and delete zones.

Animal-Employee Responsibility

This page shows which employees are responsible for caring for and training which animals. The user can add and remove responsibility relationships here.

Zone-Employee Responsibility

This page shows which employees are assigned to which zones. The user can add and remove responsibility relationships here as well.

Mixed Data

This page shows tables with data that pulls from across the database and organizes it in ways that will be useful to the zoo employees.
